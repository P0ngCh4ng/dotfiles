# Complete PON Integration Skill

## 概要

**要件入力 → 最終コード出力までワンクリック自動生成**

このスキルは、完全PON統合システムの実装方法を定義したClaude Code Skillです。DB設計からUI生成、仕様書作成、テスト生成、コード出力まで、全工程を自動化します。

## 🎯 何ができるか

- ✅ **DB PON!**: データベース設計自動生成（ER図・DDL）
- ✅ **UI PON!**: React UIコンポーネント自動生成
- ✅ **MD PON!**: Markdown仕様書自動生成（Mermaid図表対応）
- ✅ **Test PON!**: ユニット/統合テスト自動生成
- ✅ **System PON!**: 最終コード生成・ZIP出力
- ✅ **Auto PON Executor**: ワンクリック全自動実行
- ✅ **PON Flow Visualizer**: 実行フローリアルタイム可視化

### 生成コード例

```jsx
import React, { useState, useEffect } from 'react';
import { DataTable } from 'primereact/datatable';
import { Column } from 'primereact/column';
import { Button } from 'primereact/button';
import { Card } from 'primereact/card';

const UserList = ({ dataSource }) => {
  const [data, setData] = useState([]);
  const [loading, setLoading] = useState(false);

  const loadData = async () => {
    setLoading(true);
    try {
      const result = await fetch(`/api/${dataSource}`);
      const json = await result.json();
      setData(json);
    } catch (error) {
      console.error('データ読み込みエラー:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <Card title="ユーザー一覧">
      <DataTable value={data} loading={loading} paginator rows={10}>
        <Column field="id" header="ID" sortable />
        <Column field="name" header="名前" sortable />
        <Column field="email" header="メール" sortable />
        <Column
          body={(rowData) => (
            <Button label="詳細" onClick={() => console.log(rowData)} />
          )}
        />
      </DataTable>
    </Card>
  );
};

export default UserList;
```

## 🎨 アーキテクチャ

### データフロー

```
User Input (要件)
    ↓
DB PON! → dataModel
    ↓
UI PON! → uiScreens
    ↓
MD PON! → specification
    ↓
Test PON! → testCases
    ↓
System PON! → generatedCode (ZIP)
```

### Context API

全PON間でデータを共有：

```javascript
const { ponData, updatePonData } = usePonData();

// データ保存
updatePonData('dataModel', ponOutput);

// データ取得
const dbData = ponData.dataModel;
```

### 自動実行フロー

```javascript
useEffect(() => {
  const validation = validatePonOutput(ponData.dataModel, PON_TYPES.DB);

  if (validation.valid && !autoGenerated) {
    // 前PONのデータが有効なら自動実行
    generateContent();
  }
}, [ponData.dataModel]);
```
